      Agentic Financial Decision Twin

      What this build demonstrates

      Twin runs a live digital model of a user's finances — individual, startup, or enterprise — and simulates a decision's outcomes before anything is recommended. This build is the functional frontend: profile switching, a simulation engine driven by real inputs, an agent pipeline, and a data-grounded chat interface all run client-side against sample profiles, standing in for the live Data Layer until the backend is connected.
